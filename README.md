# Job Notifier Bot 🚀

Bot em .NET que monitora vagas automaticamente (Gupy) e envia notificações via Telegram.

## 🔥 Funcionalidades

- Coleta vagas de múltiplas empresas (Gupy)
- Filtro inteligente (tech + nível + exclusões)
- Evita duplicação de vagas
- Integração com Telegram

## 🛠️ Tecnologias

- .NET
- HttpClient
- JSON parsing
- Regex (Next.js data extraction)

## ⚙️ Como rodar

```bash
dotnet run
